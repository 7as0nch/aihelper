# Introduction

Welcome to the RuoYi Web documentation!

RuoYi Web is a modern web application built with Vue.js that provides chatbot capabilities with support for various AI models including ChatGPT, Midjourney, and more.

## Features

- Chatbot integration with multiple AI models
- Responsive design for desktop and mobile
- File handling capabilities
- Payment integration
- Knowledge base management
- Support for image generation with Midjourney
- Text-to-speech capabilities
