# 需求分析

随着人工智能技术的飞速发展，智能助手已在各类业务场景中得到广泛应用。本项目致力于开发一套高度可定制的智能助手系统。
其核心功能定位为：类客服形态的 AI 聊天机器人(面向管理人员，客户可以有不同功能)。
需实现以下关键能力：通过集成 MCP 服务以兼容第三方接口（第一版计划对接自主研发的基于 PostgreSQL 的 MCP 服务，该服务能够生成用户反馈内容的报表，并结合 Excel 处理 Agent 构建工作流——该工作流第一步根据用户输入描述使用 pgmcp 查询数据库并返回参数，第二步利用第一步返回的参数通过 excel 智能体为用户导出 excel 报表），最终将结果导出给用户；提供专业的管理后台，支持为 AI 助手配置可调用的 MCP 及工作流，确保 AI 助手能够依据预设工作流执行多轮接口调用，并最终向用户反馈完整的操作执行结果。

### 后续系统可考虑架构为：

1. SaaS 模式：直接订阅使用服务。
2. 私有化模式：用户自行搭建环境，并自行部署服务。(数据更安全可靠)

# 开发计划

1. 创建项目结构 【已完成】

   - 初始化项目仓库
   - 搭建项目基础框架（前端、后端、数据库）
