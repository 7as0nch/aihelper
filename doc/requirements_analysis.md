# 需求分析
随着人工智能技术的飞速发展，智能助手已在各类业务场景中得到广泛应用。本项目致力于开发一套高度可定制的智能助手系统，其核心功能定位为类客服形态的AI聊天机器人，需实现以下关键能力：通过集成MCP服务以兼容第三方接口（第一版计划对接自主研发的基于PostgreSQL的MCP服务，该服务能够生成用户反馈内容的报表，并结合Excel处理Agent构建工作流——该工作流第一步根据用户输入描述使用pgmcp查询数据库并返回参数，第二步利用第一步返回的参数通过excel智能体为用户导出excel报表），最终将结果导出给用户；提供专业的管理后台，支持为AI助手配置可调用的MCP及工作流，确保AI助手能够依据预设工作流执行多轮接口调用，并最终向用户反馈完整的操作执行结果。
