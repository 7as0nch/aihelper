# 需求分析
随着人工智能技术的飞速发展，智能助手已在各类业务场景中得到广泛应用。本项目致力于开发一套高度可定制的智能助手系统。
其核心功能定位为：类客服形态的AI聊天机器人(面向管理人员，客户可以有不同功能)。
需实现以下关键能力：通过集成MCP服务以兼容第三方接口（第一版计划对接自主研发的基于PostgreSQL的MCP服务，该服务能够生成用户反馈内容的报表，并结合Excel处理Agent构建工作流——该工作流第一步根据用户输入描述使用pgmcp查询数据库并返回参数，第二步利用第一步返回的参数通过excel智能体为用户导出excel报表），最终将结果导出给用户；提供专业的管理后台，支持为AI助手配置可调用的MCP及工作流，确保AI助手能够依据预设工作流执行多轮接口调用，并最终向用户反馈完整的操作执行结果。

# 开发计划

1. 创建项目结构 【已完成】
    - 初始化项目仓库
    - 搭建项目基础框架（前端、后端、数据库）
    - 配置项目依赖（如Vue、Node.js、PostgreSQL等）
    - 技术架构：
        - 前端：Vue 3 + Naive UI + AntV + Tailwind CSS + TypeScript
        - 后端：Golang + Kratos

        - 部署：Docker + Kubernetes

2. 基础聊天流程 【已完成】
   - 实现用户与AI助手的基本交互功能
   - 支持文本输入和输出
   - 集成MCP服务，实现与数据库的交互
